## 14. Design Reference & Inspiration

### Modern Header Design Principles

1. **Clear Visual Hierarchy**
   - Logo should be prominent but not overwhelming
   - Navigation links should be easily scannable
   - Active state should be immediately obvious

2. **Subtle Visual Effects**
   - Use shadows sparingly for depth
   - Smooth transitions (200ms ease-out)
   - Subtle hover effects (scale, lift, color change)

3. **Professional Spacing**
   - Generous padding for touch targets
   - Consistent spacing between elements
   - Proper alignment and centering

4. **Refined Active States**
   - Underline with accent color (modern, clean)
   - OR subtle background with shadow (elegant)
   - Should be clearly distinguishable

5. **Backdrop Styling**
   - Sufficient blur for separation (backdrop-blur-lg)
   - Appropriate opacity (85-95%)
   - Subtle border for definition

### Example Active State Styles

**Option 1: Underline Indicator**
```css
/* Active */
border-b-2 border-gray-900 dark:border-white
/* Hover */
hover:border-b-2 hover:border-gray-600 dark:hover:border-gray-400
```

**Option 2: Background + Shadow**
```css
/* Active */
bg-gray-900 dark:bg-white text-white dark:text-gray-900 shadow-sm
/* Hover */
hover:bg-gray-100 dark:hover:bg-gray-800
```

**Option 3: Accent Background**
```css
/* Active */
bg-gray-100 dark:bg-gray-800 border border-gray-200 dark:border-gray-700
```

### Example Logo Enhancement

**Option 1: Enhanced Typography**
```css
text-2xl font-bold tracking-tight
hover:scale-105 transition-transform duration-200
```

**Option 2: With Background**
```css
px-3 py-1.5 rounded-lg bg-gray-100 dark:bg-gray-800
hover:bg-gray-200 dark:hover:bg-gray-700
```

**Option 3: With Border**
```css
px-3 py-1.5 rounded-lg border-2 border-gray-900 dark:border-white
hover:border-gray-600 dark:hover:border-gray-400
```

### Recommended Backdrop Styling

```css
backdrop-blur-lg bg-white/95 dark:bg-black/95
border-b border-gray-200/80 dark:border-gray-800/80
shadow-sm
```
