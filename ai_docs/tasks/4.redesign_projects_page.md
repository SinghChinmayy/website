## 14. Design Reference & Industry Standards

### Modern Portfolio Project Page Principles

1. **Simple and Clean**
   - Minimal design elements
   - Generous white space
   - Clear visual hierarchy
   - Focus on content

2. **Grid-Based Layout**
   - Responsive grid system
   - Consistent card sizing
   - Good spacing between items
   - Easy to scan

3. **Card Design**
   - Subtle shadows instead of borders
   - Hover effects for interactivity
   - Clean, minimal aesthetic
   - Good padding and spacing

4. **Typography**
   - Clear hierarchy
   - Readable font sizes
   - Appropriate line heights
   - Consistent spacing

5. **Image Presentation**
   - Proper aspect ratios
   - High quality display
   - Smooth loading
   - Optional hover effects

6. **Tech Stack Display**
   - Subtle, not overwhelming
   - Small, readable tags
   - Good color contrast
   - Organized layout

### Example Grid Layout

```css
/* Mobile */
grid-cols-1

/* Tablet */
sm:grid-cols-2

/* Desktop */
lg:grid-cols-2 or lg:grid-cols-3
gap-6 sm:gap-8
```

### Example Card Design

```css
/* Base */
bg-white dark:bg-gray-900
rounded-xl
shadow-sm
p-6 sm:p-8
transition-all duration-200 ease-out

/* Hover */
hover:shadow-md
hover:-translate-y-1
```

### Example Tech Stack Tags

```css
/* Subtle, minimal */
px-3 py-1
text-xs
bg-gray-100 dark:bg-gray-800
text-gray-700 dark:text-gray-300
rounded-full
```

### Recommended Spacing

- Card padding: `p-6 sm:p-8`
- Gap between cards: `gap-6 sm:gap-8`
- Section spacing: `py-16 sm:py-20`
- Content spacing: `space-y-4` or `space-y-6`
