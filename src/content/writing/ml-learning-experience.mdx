# February 2nd

## Pandas Basics

* **View Data**: `.head()` is used to view the top 5 elements of the dataframe.
* **Selection**: Use `.loc` to select specific rows and columns based on filter criteria.
  * Syntax: `df.loc[row_selector, column_selector]`
* **DataFrame Creation**:
  * Use `pd.DataFrame` to convert data into a fresh Pandas dataframe.
  * This is essential for data processing and cleaning before visualization.

## Data Cleaning & Feature Engineering

Example dataset: [US-pumpkins.csv](https://github.com/microsoft/ML-For-Beginners/blob/main/2-Regression/data/US-pumpkins.csv)

### 1. Normalizing Data

We found inconsistent weights in the dataset, so normalization was required.

* **Using `.loc` for conditional updates**:

  ```python
  new_pumpkins.loc[new_pumpkins['Package'].str.contains('1/2'), 'Price'] = price / (1/2)
  ```

### 2. Date Processing

Extracting the month from the date column:

```python
month = pd.DatetimeIndex(pumpkins['Date']).month
```

### 3. Modifying the DataFrame

We merged the columns with the existing dataframe to create a new dataframe with only the relevant columns and new derived features:

```python
new_pumpkins = pd.DataFrame({
    'Month': month, 
    'Package': pumpkins['Package'], 
    'Low Price': pumpkins['Low Price'], 
    'Price': price
})
```
